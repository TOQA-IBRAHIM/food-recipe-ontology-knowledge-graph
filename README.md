# Food Recipe Ontology – Knowledge Graph Project

## 📌 Overview
This project presents an ontology-based system for organizing and retrieving food recipes using semantic technologies.

The ontology enables users to search for recipes based on ingredients, cooking methods, and preferences, supporting intelligent filtering and recommendation.

---

## 🎯 Objective
To design and implement a structured ontology that represents food recipes and their relationships, enabling efficient knowledge representation and semantic querying.

---

## 💡 Scenario
The system allows users to search for recipes based on specific needs.  
For example, a user can request recipes using certain ingredients (e.g., fish and vegetables) and specific cookware, and the system returns relevant results.

---

## 🧠 Key Features
- Structured representation of recipes using ontology  
- Semantic search based on ingredients, cooking methods, and meal types  
- Support for personalized recipe recommendations  
- Interoperability for integration with other applications  

---

## 🏗️ Ontology Components
- Classes (e.g., Recipe, Ingredient, MealType)  
- Object Properties (relationships between entities)  
- Data Properties (attributes such as cooking time)  
- Instances (A-Box data)  
- Class hierarchy and relationships  

---

## 🔍 Technologies Used
- OWL (Web Ontology Language)  
- SPARQL (for querying the ontology)  
- Knowledge Representation and Reasoning (KRR) concepts
